# gol2p
C++ implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) using [SDL2](https://www.libsdl.org/) with support for [WebAssembly (Wasm)](https://en.wikipedia.org/wiki/WebAssembly) via [Emscripten](https://emscripten.org/).

Go to https://arminms.github.io/gol2p to see Wasm version in action.
